# Understanding UX with Personality-Driven LLM Chatbots

## Requirements:

- ollama
- uv
- npm
- qwen2:7b downloaded through ollama

## To run the backend:

- From inside the project root run:

```
uv run main.py
```

## To run the frontend:

- From inside the bot_ui folder run the following:

```
npm install
```
```
npm run dev
```
